## 使用方式

### CLI命令行

```bash
# 手动同步
python memory_sync_service.py sync [workspace]

# 查看状态
python memory_sync_service.py status [workspace]

# 启动后台服务 (每30分钟自动同步)
python memory_sync_service.py daemon [workspace]

# 测试记录
python memory_sync_service.py record [workspace]
```

### QClaw集成

```python
from memory_sync_service import SyncCoordinator

# 初始化
sync = SyncCoordinator(workspace)

# 启动时
sync.on_startup()

# 记录对话
sync.record_conversation(
    role='user',
    content='用户消息',
    key_points=['关键点1', '关键点2'],
    decisions=['决策1']
)

# 对话结束
sync.on_conversation_end()

# 关闭时
sync.on_shutdown()
```
